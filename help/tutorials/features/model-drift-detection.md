# 模型漂移偵測

使用Mix Modeler中的模型漂移偵測功能，會自動偵測模型上的漂移，並可讓您立即採取行動或更詳細地探索漂移。 如需詳細資訊，請參閱Mix Modeler檔案中的[模型漂移](https://experienceleague.adobe.com/en/docs/mix-modeler/using/models/insights#model-drift){target="_blank"}。
