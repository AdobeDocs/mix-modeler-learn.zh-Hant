# 以目標為基礎的規劃

透過Mix Modeler中的目標型規劃功能，您可以根據業務目標進行規劃，而不只是根據預算。 您可以放心地跨管道和行銷活動有效率地分配預算，以最大的投資報酬率達成您的目標。
