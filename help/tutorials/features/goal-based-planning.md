# 目標型規劃

透過Mix Modeler中的目標型規劃功能，您可以根據業務目標進行規劃，而不只是根據預算。 您可以放心地跨管道和行銷活動有效率地分配預算，以最大的投資報酬率達成您的目標。 如需詳細資訊，請參閱Mix Modeler檔案中的[計畫](https://experienceleague.adobe.com/zh-hant/docs/mix-modeler/using/plans/overview){target="_blank"}。
